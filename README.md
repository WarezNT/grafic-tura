# Grafic de Tură

Aplicație web pentru generarea graficelor de ture (T1, T2, T1A, T2A, 8h) pentru grupe Urban și Rural.

Un singur fișier `index.html` — fără instalare, fără server. Datele se salvează local în browser (localStorage + IndexedDB).

**Demo:** https://wareznt.github.io/grafic-tura/

## Utilizare

Deschide `index.html` în Chrome sau Edge.
